# QRKot - Благотворительный фонд поддержки котиков

## Описание проекта

QRKot — это API-приложение для благотворительного фонда поддержки котиков. 
Фонд собирает пожертвования на различные цели: корм для бездомных кошек, 
медицинское обслуживание, организацию приютов и другую помощь кошачьей популяции.

### Основные возможности

- **Управление целевыми проектами**: создание, редактирование, удаление и просмотр 
  проектов с определённой суммой сбора.
- **Пожертвования**: пользователи могут делать пожертвования с комментариями.
- **Автоматическое инвестирование**: пожертвования автоматически распределяются 
  по открытым проектам в порядке их создания (FIFO).
- **Прозрачность**: каждый проект и пожертвование имеют полную историю 
  с датами создания и закрытия.

### Технологии

- **FastAPI** — современный веб-фреймворк для Python
- **SQLAlchemy** — ORM для работы с базами данных
- **Alembic** — миграции базы данных
- **Pydantic** — валидация данных
- **SQLite** — база данных (для разработки)
- **Uvicorn** — ASGI-сервер
